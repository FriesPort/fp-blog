## tar 命令

### 常用

| 选项   | 解释              |
| ---- | --------------- |
| `-c` | 创建包             |
| `-x` | 解开包             |
| `-f` | 指定包的文件名         |
| `-v` | 展示操作的过程         |
| `-z` | 调用gzip进行压缩和解压   |
| `-C` | 执行任何操作前先切换到指定目录 |

### 高级

| 选项   | 解释      |
| ---- | ------- |
| `-t` | 列出包内的文件 |
| `-u` | 更新文件    |

## tar 案例

### 打包指定目录到文件

```bash
tar -cvf pack.tar -C /path/to/parent target-dir 
```

打包目录为 `/path/to/parent/target-dir` 包内没有父目录，结构干净。还可以搭配`-z`调用gzip对归档包进行压缩。

具体含义如下：`-c`创建一个归档包，`-v`显示具体操作，`-f`指定压缩包名字。`-C`在执行具体命令之前先切换到目标目录。

### 解包指定文件到目录

```bash
tar -xvf pack.tar -C /path/to/extract
```

解压后目录为 `/path/to/extract/pack` 。

具体含义如下：`-x`解开一个压缩包，`-v`显示具体操作，`-f`指定压缩包名字。`-C`在执行具体命令之前先切换到目标目录。

## zip 命令

zip需要额外安装，会一并安装unzip。

| 选项    | 解释  |
| ----- | --- |
| `-r`  | 递归  |
| `​-q` | 安静  |
| `-e`  | 加密  |

指定目录，类似tar的 `-C` 命令

## zip 案例

### 压缩

```bash
zip -rq pack.zip /path/to/target
```

打包目录为 `/path/to/target` 。

### 解压

```bash
unzip pack.zip -d /path/to/extract
```

得到 `/path/to/extract/pack` 目录。
